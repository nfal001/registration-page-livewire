## Installation

Composer Install Project

```
composer install
```

Copy Enviroment file

```
cp .env.example .env
```

Generate App Key

```
php artisan key:generate
```

Export Migration

```
php artisan migrate
```

seed database

```
php artisan db:seed
```

Start Server

```
php artisan serve
```


## Endpoint
- `/interview` - CounterTest
- `/daftar-santri` - Forms And Cek Kuota Santri
